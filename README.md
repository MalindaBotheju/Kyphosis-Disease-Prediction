# Kyphosis Prediction: End-to-End Machine Learning Web App

A medical screening tool that predicts the presence of Kyphosis after spinal surgery.

## 📊 Model Performance
[cite_start]Our Tuned SVM model was optimized for medical safety[cite: 437, 584]:
* [cite_start]**Recall (Medically Important):** 75% for 'Present' cases[cite: 441, 589].
* [cite_start]**Accuracy:** 76%[cite: 441, 589].
* [cite_start]**Dataset:** 81 patient records[cite: 305, 452].

## 🚀 How to Run Locally
1. Clone this repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `python app.py`
